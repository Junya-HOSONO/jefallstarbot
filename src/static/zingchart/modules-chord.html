JSON Attributes/Syntax

options 

Main object for definitions and styling directives.
  Attribute Type Info
angle-padding
anglePadding		 Numeric

Sets the angle padding between two chord segments.

5 | 10 | ...
band-space
bandSpace		 Numeric

Sets the space between the band and the chords.

5 | 10 | ...
band-width
bandWidth		 Numeric

Sets the width of the band.

5 | 10 | ...
color Mixed

Sets the color of the items when color-type is set to color.

"none" | "transparent" | "#f00" | "#f00 #00f" | "red yellow" | "rgb(100, 15, 15)" | ...
color-type
colorType		 String

Sets the color algorithm used to color the items.

"random" | "palette" | "color"
palette Array

Defines a custom palette used by the chord if color-type is set to palette.

["#f90", "#0ff", ...]
radius Numeric

Sets the radius of the chord. Value can be relative to the plotarea size.

5 | 10 | ...
static Boolean

Sets whether the chord "highlight" effect when moving over the band is enabled or not.

true | false | 1 | 0
style Object

Defines global styling for chord elements.
