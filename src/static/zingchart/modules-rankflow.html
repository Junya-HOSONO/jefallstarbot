JSON Attributes/Syntax

options 

Main object for definitions and styling directives.
  Attribute Type Info
col-space
colSpace		 Numeric

Sets (forces) the size of the spacing between flow columns.

5 | 10 | ...
color Mixed

Sets the color of the items when color-type is set to color.

"none" | "transparent" | "#f00" | "#f00 #00f" | "red yellow" | "rgb(100, 15, 15)" | ...
color-type
colorType		 String

Sets the color algorithm used to color the items.

"random" | "palette" | "color"
palette Array

Defines a custom palette used by the rankflow if color-type is set to palette.

["#f90", "#0ff", ...]
row-space
rowSpace		 Numeric

Sets (forces) the size of the spacing between flow rows.

5 | 10 | ...
sep-space
sepSpace		 Numeric

Sets (forces) the size of the spacing between the overall section and the flow section.

5 | 10 | ...
static Boolean

Sets whether the main labels are active or not, allowing for one or more flow selection.

true | false | 1 | 0
style Object

Defines global styling for rankflow elements.
