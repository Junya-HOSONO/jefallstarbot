ZingChart Build Generator

The Build Generator tool allows customers to create a custom build containing only modules that are needed for a specific task/project. This is an alternative to the dynamic load of modules, in order to avoid multiple HTTP requests.

The Build Generator allows customers to pick specific modules which will be compacted in a single file called zingchart-html5-min.js. In addition, the zip package that is generated contains a configuration file (zingchart-html5.cnf) containing the list of included modules. This allows customers to easily get later builds including the same modules by only uploading the configuration file in the Build Generator.
