WordCloud Module

A tag cloud (word cloud, or weighted list in visual design) is a visual representation for text data, typically used to depict keyword metadata (tags) on websites, or to visualize free form text. 'Tags' are usually single words, and the importance of each tag is shown with font size or color. This format is useful for quickly perceiving the most prominent terms and for locating a term alphabetically to determine its relative prominence. When used as website navigation aids, the terms are hyperlinked to items associated with the tag.

 

Data Format

  

{
	"graphset" : [
		{
			"type" : "wordcloud",
			"options" : {
				"text" : "...",
				"ignore" : [...],
				"max-items" : ...,
				"min-length" : ...,
				"token" : "...",
				"min-font-size" : ...,
				"max-font-size" : ...,
				"color-type" : "...",
				"color" : "...",
				"rotate" : ...,
				"palette" : [...],
				"step-angle" : ...,
				"step-radius" : ...,
				"style" : {
					...,
					"hover-state" : {
						...
					},
					"tooltip" : {
						...
					}
				}
			}
		}
	]
}

 

JSON Attributes/Syntax

options 

Main object for definitions and styling directives.
  Attribute Type Info
aspect String

Sets the layout algorithm.

"spiral" | "flow-center" | "flow-top"
color Mixed

Sets the color of the tokens used when color-type is set to color.

"none" | "transparent" | "#f00" | "#f00 #00f" | "red yellow" | "rgb(100, 15, 15)" | ...
color-type
colorType		 String

Sets the color algorithm used to color the items.

"random" | "palette" | "color"
ignore Array

Sets the words that will be ignored. Used to eliminate common words that have no relevance and may alter the output.

[...]
max-font-size
maxFontSize		 Numeric

Sets the maximum size of the fonts used when scaling and laying out the tokens. The value may be changed by the layout engine if tokens do not fit inside the plotarea.

5 | 10 | ...
max-items
maxItems		 Numeric

Sets the maximum number of tokens displayed.

5 | 10 | ...
min-font-size
minFontSize		 Numeric

Sets the minimum size of the fonts used when scaling and laying out the tokens.

5 | 10 | ...
min-length
minLength		 Numeric

Sets the minimum length of the words being analyzed.

5 | 10 | ...
palette Array

Defines a custom palette used by the treemap if color-type is set to palette.

["#f90", "#0ff", ...] | [{...}, {...}, ...]
rotate Boolean

Sets if one in two tokens is being rotated by 90 degrees.

true | false | 1 | 0
step-angle
stepAngle		 Number

Sets the step of the angle used by the layout engine when placing items along a spiral.

5 | 10 | ...
step-radius
stepRadius		 Number

Sets the step of the radius used by the layout engine when placing items along a spiral.

5 | 10 | ...
style Object

Defines global styling for wordcloud elements.

{...}
text String

Sets the text that will be analyzed.

"..."
token String

Sets the type of tokens being analyzed.

"word" | "character"
