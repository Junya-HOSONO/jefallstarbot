Pareto Module

The Pareto chart, is a type of chart that contains both bars and a line graph, where individual values are represented in descending order by bars, and the cumulative total is represented by the line. The left vertical axis is the frequency of occurrence, but it can alternatively represent cost or another important unit of measure. The right vertical axis is the cumulative percentage of the total number of occurrences, total cost, or total of the particular unit of measure.

 

Data Format

  

{
	"graphset" : [
		{
			"type" : "pareto",
			"options" : {
				"scale-y-2" : {
					...
				},
				"line-plot" : {
					...
				}
			},
			"series" : [
				{
					...
				}
			]
		}
	]
}
